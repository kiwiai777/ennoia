# Cortex 核心价值（精简公开版）

## 一句话

Cortex 的核心价值不是“记住更多内容”，而是：

**让 AI 系统真正认识同一个用户。**

## Cortex 解决的问题

今天的大多数 AI 工具都存在同一个问题：

- 每个 agent 都像第一次认识用户
- 用户需要反复解释自己
- 偏好、约束、目标、做事方式无法稳定跨工具继承
- 即使有 memory，也常常只是片段存储，而不是用户理解

Cortex 要解决的不是“更多记忆”，而是：

**用户在 AI 世界中缺乏持续存在的问题。**

## Cortex 的核心能力

Cortex 希望把原始输入（文件、对话、workspace、导入数据等）逐步沉淀成用户模型，让 AI 工具共享同一个“我”。

核心包括：

1. **结构化用户理解**
   - goal
   - project
   - preference
   - constraint
   - skill
   - decision rule
   - state

2. **持续存在**
   - 用户模型可以跨会话存在
   - 不依赖某一轮 prompt
   - 不依赖某一个单独 agent

3. **跨 agent 一致性**
   - 不同 AI 工具可以逐步共享同一份用户理解
   - 让多个 agent 看起来都“认识同一个人”

4. **行为影响**
   - 不只是展示上下文
   - 还要逐步影响 agent 的输出与行为方式

## 为什么这不是普通 memory

普通 memory 更关注：

- 记了什么
- 能不能找回来
- 召回是否相关

Cortex 更关注：

- 这些信息是否构成稳定的用户模型
- 是否能区分长期特征和临时片段
- 是否能给 agent 提供可持续的行为信号

所以 Cortex 的目标不是“拥有更多原文”，而是：

**把用户相关信息编译成能影响 AI 行为的用户理解层。**

## 为什么这很重要

如果 AI 只会回答问题，却不真正理解用户，那么用户和 AI 之间的关系始终是脆弱的、一次性的。

Cortex 的长期价值在于：

- 减少用户重复解释自己的成本
- 让 AI 行为更贴近用户
- 让用户在不同 AI 系统中成为一等实体
- 让“User Model”成为新的基础层
